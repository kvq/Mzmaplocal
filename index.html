About
This Minecraft web map is part of the MineMap Project and was independently developed by RedBanHammer. MineMap began with the simple goal to create the best online map possible for the vast world of MineZ. Since then, the project has expanded to many other Minecraft video game servers.
Special thanks to Andrew323, Kittykiller, Xeroxcat, Majicou, Mosh_Von_Void, IvanDoomy, Fridge and many others for contributing to this project.
The Minecraft world render was created using the Minecraft Overviewer, the map is powered by Leaflet, and the interface uses Bootstrap and Font Awesome.

    Export Waypoints Help

    1. Type in the server you normally join (i.e. us.minez.net)
    2. Select the types of waypoints you want (i.e. no spawn, only major, etc)
    3. Click Download, and locate the downloaded SERVERNAME.DIM0.points file
    4. Drag the file into either...
      %Appdata%/.minecraft/mods/rei_minimap (Windows)
      /Users/USER/Library/Application Support/minecraft/mods/rei_minimap (Mac)
    5. Open Minecraft. If you didn't select for waypoints to be visible by default, press [m], click Waypoints, and enable any you need.
